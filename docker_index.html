4. Docker 活用記録

Dockerを使って以下のような環境を構築しました。

  • WordPress + MariaDB を Docker Compose で構築
  • 自動起動設定やボリュームマウントによる永続化設定

起動中のコンテナ一覧（CLI出力）

$ docker ps
CONTAINER ID   IMAGE               COMMAND                  STATUS        NAMES
abcd1234       wordpress:latest    "docker-entrypoint.s…"   Up 3 hours    wp_app
bcde2345       mariadb:latest      "docker-entrypoint.s…"   Up 3 hours    wp_db

今後改善したいポイント

  • ボリュームのバックアップ自動化（cron + rsync）
  • ログの永続化と集約（Fluentd または Lokiなど）
  • 開発・本番での環境差分吸収のための.env設計
